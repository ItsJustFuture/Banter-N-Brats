## Funny/Comedic Events

**Players:** 1-30  
**DC:** 13 (Instinct)

## Content Guidelines

All events follow these principles:
- **Funny**: Lighthearted, slapstick, harmless comedy
- **Romantic**: Sweet, heartfelt, emotionally resonant
- **Sexy**: Tasteful, appropriate for 18+, suggestive not explicit
- **Gory**: Intense but not gratuitous, mature horror themes
- **Game-Altering**: Significant mechanical impact on gameplay
- **Challenge**: Test player strategy and character builds
- **Mysterious**: Intrigue, unknown consequences, risk/reward

## Running the Game

### For Game Masters:
1. Start a DnD session as usual
2. System automatically detects couples
3. Advance rounds - events are selected randomly
4. Couple events appear more often when couples present
5. Watch for the ðŸ’• indicator on couple events

### For Players:
1. Create your character with attributes and skills
2. Join the session
3. If you're in a couple, enjoy the bonuses!
4. React to events as they unfold
5. Survive and thrive through the adventure
